
## 📌 `/google` [GET]

Endpoint to redirect users to the **Google OAuth2 login page**.

---

### ✅ Request

- **Method**: `GET`
- **Query**: *None*
- **Headers**: *None*

---

### 🔸 Example (JavaScript - `window.location`)

```
window.location.href = '/google';

```

---

### ✅ Response

### ▶ Redirect – Google OAuth2 Login Page (HTTP 302)

- Redirects the client to the Google OAuth2 authorization URL.

### ▶ Server Error – Internal Failure (HTTP 500)

```json
{
  "success": false,
  "msg": "An error occurred during Google login redirection."
}

```

---

### 🧩 Additional Notes

- **Rate Limiting**: Max 5 requests per minute.
- **Redirect Logic**: URL generated by `googleOAuth2Manager.getGoogleOAuthUrl()`.
